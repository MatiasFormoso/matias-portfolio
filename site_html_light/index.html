Casos

Market Indicators Dashboard

Herramienta web gratuita para seguimiento y análisis de activos financieros (cripto y mercado), con ingesta automática vía GitHub Actions, almacenamiento en PostgreSQL (Neon) y visualización en una app Streamlit desplegada en Render. Controles de rango, granularidad y normalización (base 100), además de KPIs y velas OHLC.

  • Stack: Python, Streamlit, PostgreSQL (Neon), GitHub Actions, Render, APIs de datos (FMP / CoinGecko).
  • Arquitectura: Ingesta programada → validación/transf. → tablas precios e historial → app.
  • Operación continua: contenedor Docker, env vars, healthchecks y cron jobs vía Actions.

GitHub Actions (ingesta) → Neon Postgres → Streamlit en Render. Variables y Docker.

Pipeline MP4 end-to-end a partir de texto

De un .txt a MP4, completamente automático: narración TTS, selección de clips por IA, transiciones suaves, mezcla de música con ducking, subtítulos estilizados y limpieza automática. Arquitectura modular por etapas y control de errores con reintentos.

  • Stack: Python (MoviePy, ffmpeg, pydub), Whisper, ElevenLabs, API de Pexels, Streamlit, Mistral/Ollama.
  • Retos: sincronización audio-video, subtítulos limpios, cobertura 1:1 del audio, uso eficiente de RAM.
  • Soluciones: procesamiento por lotes, lógica por fragmentos con reintentos, queries por IA local, ducking y fades compensados, limpieza automática.
  • Estado: MVP funcional con logs claros, configuración por entorno y módulos intercambiables.

Entrada .txt → TTS/segmentación → búsqueda de clips → ensamblado → subtítulos → render final.
