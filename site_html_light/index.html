Casos

Market Indicators Dashboard

Herramienta web gratuita para seguimiento y análisis de activos financieros (cripto y mercado), con ingesta automática vía GitHub Actions, almacenamiento en PostgreSQL (Neon) y visualización en una app Streamlit desplegada en Render. Controles de rango, granularidad y normalización (base 100), además de KPIs y velas OHLC.

  • Stack: Python, Streamlit, PostgreSQL (Neon), GitHub Actions, Render, APIs de datos.
  • Arquitectura: Ingesta programada → validación/transf. → tablas precios e historial → app.
  • DevOps: Docker, variables de entorno y healthchecks. Operación continua con cron jobs vía Actions.
  • Resultados: latencia < 2 s, cero intervenciones diarias, datos normalizados comparables.

Pipeline MP4 end-to-end a partir de texto

De un .txt a MP4, completamente automático: narración TTS, selección de clips por IA, transiciones suaves, mezcla de música con ducking, subtítulos estilizados y limpieza automática.

  • Stack: Python (MoviePy, ffmpeg, pydub), Whisper, ElevenLabs, API de Pexels, Streamlit.
  • Retos: sincronización audio-video, subtítulos limpios, cobertura 1:1, uso eficiente de RAM.
  • Soluciones: procesamiento por lotes, lógica por fragmentos con reintentos, IA local, ducking y fades.
  • Resultados: render estable < 90 s para piezas cortas, subtítulos sincronizados y pipeline reejecutable.
