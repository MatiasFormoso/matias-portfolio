Agente de búsqueda laboral — automatización con IA

Pipeline Sabueso → Sastre → Cartero: descubre avisos, puntúa fit, genera CV y mail personalizados con Gemini, y gestiona envíos con cola de aprobación humana.

PythonGeminiStreamlit SQLitePlaywrightGmail API CursorScraping

Contexto

Buscar trabajo de forma manual implica revisar decenas de avisos, adaptar CV y redactar mails uno por uno. Construí un agente personal que automatiza la parte repetitiva pero mantiene control humano antes de cada envío.

Estado del proyecto

  • ▶ En uso personal — funcional para búsqueda activa de empleo.
  • ▶ No es un producto comercial ni SaaS público.
  • ▶ Desarrollo intensivo asistido con Cursor e IA.

Arquitectura (3 módulos)

  • ▶ Sabueso: scraping y filtrado de avisos (LinkedIn, GetOnBoard, etc.) con reglas de descarte (geo, modalidad, keywords).
  • ▶ Sastre: scoring de fit + generación de CV HTML y mail con Gemini, usando catálogo de proyectos reales como fuente.
  • ▶ Cartero: cola de aprobación, export PDF (Playwright) e integración Gmail para envío.

Qué hice

  • ▶ UI Streamlit con pestañas de revisión, exploración y barras de progreso.
  • ▶ Base SQLite con historial de jobs, scores y paquetes generados.
  • ▶ CV tailoring que reemplaza sección completa de proyectos según el aviso (no solo el resumen).
  • ▶ Catálogo YAML de proyectos como fuente única de verdad para el Sastre.
  • ▶ PDF export con Playwright (Chrome/Edge fallback) y browsers embebidos en el repo.
  • ▶ Filtros agresivos: descarte de híbrido/presencial fuera de Argentina, fiscal Chile, etc.

Stack

Python 3.12 · Google Gemini · Streamlit · SQLite · Playwright · Gmail API · YAML · Cursor IDE
