Cómo verlo local (sin Git)
1) Abrí esta carpeta y hacé doble clic en index.html
2) Opcional: en Terminal -> python -m http.server 5500 -> http://localhost:5500

Deploy en Vercel (producción)
1) cd site
2) vercel link          (si no está linkeado al proyecto matiasformoso)
3) vercel --prod        (publica en https://matiasformoso.vercel.app)
