Sobre mí

Estudiante avanzado de Ingeniería en Sistemas (UTN-FRC, Argentina). Desarrollo Full-Stack con foco en entregas reales: sitios para clientes en producción, prototipos de APIs e integraciones, y proyectos propios en Python y datos.

Trabajo con Cursor + IA de forma productiva: no reemplazo el criterio técnico, lo acelero. Entiendo arquitectura, debugging y trade-offs — la IA es una herramienta, no un atajo para vender humo.

  • Full-Stack web · Next.js, TypeScript, entregas para clientes internacionales
  • Backend & APIs · Python, NestJS/Hono, REST, webhooks, integraciones fiscales
  • Datos & automatización · ETL, Streamlit, pipelines con IA, agentes personales

Cómo trabajo

  • ▶ Cursor + IA con criterio: uso agentes para acelerar implementación, tests y refactors; reviso cada diff y entiendo lo que shippeo.
  • ▶ Honestidad técnica: diferencio entregas a cliente, prototipos/MVP y proyectos discontinuados — sin inflar métricas ni inventar tracción.
  • ▶ Entrega end-to-end: desde modelado y API hasta deploy (Vercel, Render, Docker) y documentación mínima útil.
  • ▶ Integraciones como core skill: webhooks idempotentes, manejo de errores de terceros, ambientes sandbox/producción.
